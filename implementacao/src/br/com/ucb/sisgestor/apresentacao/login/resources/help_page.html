Ajuda da tela de entrada

A Tela de entrada é usada para a autenticação do usuário no SISBACEN e é mostrada sempre que algum recurso protegido é solicitado por um usuário ainda não autenticado. Além disso, o usuário também poderá utilizar esta tela para trocar sua senha no SISBACEN.



  • Usuários SISBACEN: acesso com operador
  • Pessoas físicas: acesso com CPF
  • Procedimento de troca de senha

Usuários SISBACEN: acesso com operador

Usuários SISBACEN deverão executar os seguintes passos para se autenticar:

  1. Digitar sua unidade no campo "Unidade";
  2. Digitar sua dependência no campo "Dependência" (opcional);
  3. Digitar seu operador no campo "Operador";
  4. Digitar sua senha no campo "Senha";
  5. Pressionar o botão "Entrar".
Se o processo de autenticação for realizado com sucesso, o recurso solicitado será mostrado.


Pessoas físicas: acesso com CPF

Pessoas físicas deverão executar os seguintes passos para efetuar o acesso com CPF:

  1. Digitar os primeiros 9 dígitos de seu CPF no campo "CPF";
  2. Digitar sua senha no campo "Senha";
  3. Pressionar o botão "Entrar".
Se o processo de autenticação for realizado com sucesso, o recurso solicitado será mostrado.


Procedimento de troca de senha

A tela de troca de senha é mostrada quando o usuário acessa o link "Alterar senha" presente na tela de entrada ou quando tenta se autenticar no SISBACEN e sua senha está vencida.

Para efetuar a troca de senha, o usuário deverá executar os seguintes passos:

  1. Informar seus dados de identificação:
    • Usuários SISBACEN devem informar a unidade, a dependência (opcional) e o operador;
    • Pessoas físicas devem informar os primeiros 9 dígitos do CPF.
  2. Digitar a senha atual no campo "Senha";
  3. Digitar a senha nova no campo "Senha nova";
  4. Re-digitar a senha nova no campo "Confirmação de senha nova";
  5. Pressionar o botão "Confirmar".
Se a troca de senha for realizada com sucesso, o usuário será automaticamente autenticado no SISBACEN, já com a nova senha, e o fluxo será direcionado para a aplicação. O procedimento de troca de senha pode ser cancelado, bastando acionar o botão "Cancelar".
